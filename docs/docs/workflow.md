# Workflow du projet

## Étape 1 : Installation du QR code

Chaque chaudière reçoit un QR code unique relié à son identité numérique.

## Étape 2 : Intervention du technicien

Le technicien scanne le QR code avec son téléphone.

## Étape 3 : Rapport d’entretien

Le technicien remplit :
- date
- type d’intervention
- remarques
- pièces remplacées
- photos éventuelles

## Étape 4 : Validation blockchain

Le rapport est validé et enregistré sur la blockchain.

## Étape 5 : Consultation

Le propriétaire, la régie ou le syndic peuvent consulter l’historique des interventions.

## Étape 6 : Archivage sécurisé

Les rapports restent accessibles et infalsifiables.
