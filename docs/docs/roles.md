# Rôles du système

## Administrateur société

- Autorise les techniciens
- Gère les tablettes
- Consulte les historiques
- Désactive les accès

## Technicien

- Scanne le QR code
- Remplit le rapport
- Ajoute les photos
- Valide l’intervention

## Propriétaire

- Consulte l’historique
- Vérifie les entretiens
- Accède aux rapports

## Régie / Syndic

- Consulte les interventions
- Vérifie la conformité
- Suit les entretiens collectifs

## Blockchain

- Enregistre les validations
- Garantit l’intégrité des données
- Horodate les interventions
