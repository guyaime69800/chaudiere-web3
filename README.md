# Chaudiere-Web3

Projet de suivi d’entretien de chaudière utilisant la blockchain et les QR codes.

## Objectif

Créer un système transparent et infalsifiable permettant de suivre les entretiens de chaudières individuelles ou collectives.

## Fonctionnement

- QR code unique sur la chaudière
- Historique des interventions
- Validation des entretiens via blockchain
- Consultation par propriétaire, régie ou technicien
- Stockage sécurisé des rapports

## Technologies envisagées

- Polygon
- Solidity
- GitHub
- MetaMask
- IPFS
- QR Codes

## Vision du projet

Apporter plus de confiance et de transparence dans le domaine de la maintenance et du chauffage grâce aux technologies Web3.
